# StudyPython - 파이썬 학습 프로그램

076923.github.io의 파이썬 강좌를 기반으로 한 통합 학습 환경을 제공하는 WPF 데스크톱 애플리케이션입니다.

## 주요 기능

- **강좌 학습**: 46개 파이썬 강좌의 본문, 이미지, 코드 예제를 앱 내에서 직접 열람
- **코드 에디터**: AvalonEdit 기반 파이썬 전용 에디터 (구문 강조, 줄번호, 다중 탭 지원)
- **코드 실행**: 에디터 및 예제 코드를 즉시 실행하고 결과 확인
- **REPL**: 대화형 파이썬 셸 내장
- **PDF 보고서**: 코드와 실행 결과를 PDF로 출력 (학번_이름_날짜.pdf)
- **Git 제출**: PDF 보고서를 지정 저장소에 자동 push

## 요구 사항

- Windows 10 이상
- .NET 8.0 Runtime
- Python 3.x (시스템 PATH에 등록 필요)

## 설치 및 실행

### Release 다운로드
1. [Releases](https://github.com/samcho93/kutStudyPython/releases) 페이지에서 최신 버전 다운로드
2. 압축 해제 후 `StudyPython.exe` 실행

### 소스 빌드
```bash
git clone https://github.com/samcho93/kutStudyPython.git
cd kutStudyPython
dotnet build -c Release
```

## 사용법

### 화면 구성

```
+-------------------------------------------------------------------------+
| 학과 |          | 학번  |          | 학년  |            | 이름 |        |
+-------------------------------------------------------------------------+
| 소속 회사 |                        | 점수  |            |   [작업폴더]  |
+-----------+--------------+----------------------------------------------+
| 강좌      | 본문 내용    | 파이썬 에디터 (탭)                           |
| 리스트    |              | [NEW][OPEN][SAVE] [시작][정지]               |
|           +--------------+----------------------------------------------|
|           | 예제 |결과   | 실행 결과              [지우기][출력][제출]   |
|           +--------------+----------------------------------------------|
|           | 실행 결과    | REPL                                         |
+-----------+--------------+----------------------------------------------+
```

### 학생 정보 입력
- 상단에 학과, 학번, 학년, 이름, 소속 회사를 입력합니다.
- 최초 입력 후 자동 저장되며, 재실행 시 자동으로 불러옵니다.

### 강좌 학습
1. 왼쪽 **강좌 리스트**에서 학습할 강좌를 선택합니다.
2. 중앙 **본문 내용** 영역에 강좌 내용이 이미지와 함께 표시됩니다.
3. 본문을 스크롤하면 해당 위치의 예제가 자동으로 선택됩니다.

### 예제 실행
1. 본문 하단의 **예제** 영역에서 콤보박스로 예제를 선택합니다.
2. **실행** 버튼을 클릭하면 오른쪽 실행 결과 영역에 결과가 출력됩니다.

### 파이썬 에디터
- **NEW**: 새 파일 생성 (새 탭 추가)
- **OPEN**: 기존 .py 파일 열기 (복수 선택 가능)
- **SAVE**: 현재 탭의 코드를 작업폴더에 저장
- **시작**: 현재 탭의 코드를 실행
- **정지**: 실행 중인 코드를 중지
- 탭의 **x** 버튼으로 탭을 닫을 수 있습니다.

### 작업폴더
- **작업폴더** 버튼으로 파이썬 파일이 저장될 폴더를 지정합니다.

### REPL
- 하단 입력란에 파이썬 코드를 입력하고 Enter를 누르면 즉시 실행됩니다.

### 보고서 출력 및 제출
- **출력**: 에디터 코드와 실행 결과를 PDF 파일로 저장합니다.
- **제출**: PDF 보고서를 Git 저장소에 자동으로 push합니다.
